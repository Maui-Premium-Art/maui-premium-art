# Constitution — Maui Premium Art Website

*Established: March 12, 2026*

## Core Principles

### 1. The Site IS the Truck
The website is not a marketing site that references the Cybertruck — it IS the Cybertruck's console UI. When a CT owner opens mauipremiumart.com, they should feel like they just powered on their truck. Every design decision flows from this.

### 2. Art First, Always
The artwork is the hero. UI elements exist to frame the art, not compete with it. Generous space, restraint in chrome, let the Mahalo Bird breathe.

### 3. Premium Through Subtlety
No flashy animations, no gradient blobs, no AI-template aesthetics. Premium means: intentional negative space, pixel-perfect detail, clean typography, purposeful motion. If something feels loud, remove it.

### 4. Hawaiian Soul in a Futuristic Vessel
The visual language is CT/industrial. The spirit is Hawaiian — unhurried, warm, rooted. These don't conflict; they complete each other. Hawaiian Space Travelers.

### 5. Mobile First, No Exceptions
Most visitors come from X (mobile). The experience must be excellent on a phone before we think about desktop. Scale up gracefully, never down awkwardly.

## Guiding Constraints (v1)
- No feature ships before the shell is solid
- One art piece in the hero at a time
- Static before interactive
- No accounts, no store, no gamification yet
- Performance > polish (Core Web Vitals matter)
